### 2. GL Auto-posting (Автопроводки по двойной записи) - 2%
**Текущее состояние:** Таблицы GL есть, но автопроводки не реализованы  
**Требуется:** Автоматическое создание journal entries при проведении документов

#### Задачи:
- [ ] Создать GL posting rules:
  - [ ] ПКО: Дт 100 (Каса) / Кт 400 (Дохід)
  - [ ] РКО: Дт 500 (Витрати) / Кт 100 (Каса)
  - [ ] Рахунок (bill): Дт 500 / Кт 200 (Кредит. заборг.)
  - [ ] Оплата рахунку: Дт 200 / Кт 100
  - [ ] Членство: Дт 100 / Кт 400
- [ ] Обновить documentService.post():
  - [ ] Генерация journal entries
  - [ ] Транзакционная целостность (rollback при ошибке)
- [ ] UI для просмотра проводок по документу
- [ ] Audit trail для всех GL операций

**Примерное время:** 4-6 часов
