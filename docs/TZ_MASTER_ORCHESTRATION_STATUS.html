YBC ERP — Полное ТЗ + Оркестрация + Статус

Версия: 1.0
Дата: 2026-03-01
Стек: Supabase + Cloudflare (без Railway)

1) Цель

  • Memberships
  • Events & Tickets
  • Bills & Payments (accrual)
  • GL (двойная запись)
  • Журнал документов
  • ПКО/РКО
  • Обмен валют
  • Перемещения между кассами/кошельками
  • Sandbox + demo data + reset

2) Архитектура

  • Frontend: React + Vite
  • Data/Auth: Supabase (Postgres + Auth + RLS)
  • Hosting: Cloudflare Pages (erp.ybc.com.cy)
  • Railway: не используется в целевом варианте

3) Оркестрация

PLAN -> BUILD -> QA -> FIX -> DONE

4) Блок-схемы

Auth

User -> /login -> Supabase signInWithPassword
    -> force_password_change=true ? /change-password : /dashboard

GL (целевая)

Document (Bill/PKO/RKO/Exchange/Transfer)
  -> balanced entries Dt/Kt
  -> document journal + journal entries
  -> debit == credit check
